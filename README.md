## CARA INSTALL DI TERMUX
```bash
> pkg install nodejs && pkg install git
> git clone https://github.com/naisayuriza/line-bot
> cd line-bot
> bash install.sh
> npm start/node main.js
```
## CARA INSTALL DI LAPTOP
```bash
> git clone https://github.com/FALINE-BOT/line-bot
> cd line-bot
> npm i
> npm start/node main.js
```
